Detailed Description

template<typename T = Card>
class Deck< T >

Template class representing a deck of cards.

Template Parameters
TCard type that comprises the deck (defaults to Card class) Implements a deck of cards with shuffle, draw and tracking functionality. Supports multiple deck initialization and duplicate card detection.
